📋 Como criar a tabela no Supabase

1. Acesse o painel do Supabase → SQL Editor
2. Cole e execute o SQL abaixo:

CREATE TABLE IF NOT EXISTS turno_data (
  key text PRIMARY KEY,
  value jsonb,
  updated_at timestamptz DEFAULT now()
);

-- Permitir acesso público (anon key)
ALTER TABLE turno_data ENABLE ROW LEVEL SECURITY;
CREATE POLICY "allow_all" ON turno_data
  FOR ALL USING (true) WITH CHECK (true);
